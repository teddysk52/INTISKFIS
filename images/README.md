# Инструкция по изображениям

Для правильной работы сайта нужно добавить следующие изображения в папку `images/`:

## Обязательные изображения:

1. **hero-bg.jpg** - Фоновое изображение для главного экрана (рекомендуемый размер: 1920x1080px)
   - Можно использовать фото современного принтера или офисного пространства
   - Источник: Unsplash.com или Pexels.com (бесплатные стоковые фото)

2. **printer-dot-matrix.jpg** - Фото игольчатого принтера (800x600px)
3. **printer-inkjet.jpg** - Фото струйного принтера (800x600px)
4. **printer-laser.jpg** - Фото лазерного принтера (800x600px)
5. **printer-thermal.jpg** - Фото термопринтера (800x600px)

6. **connection-usb.png** - Иконка/фото USB подключения (400x400px)
7. **connection-wifi.png** - Иконка/фото Wi-Fi подключения (400x400px)
8. **connection-ethernet.png** - Иконка/фото Ethernet подключения (400x400px)
9. **connection-bluetooth.png** - Иконка/фото Bluetooth подключения (400x400px)

10. **office-network.jpg** - Схема офисной сети с принтером (1200x800px)

## Где взять изображения:

- **Unsplash.com** - бесплатные фото высокого качества
- **Pexels.com** - бесплатные стоковые фотографии
- **Pixabay.com** - бесплатные изображения и векторная графика
- **Google Images** - поиск с фильтром "Разрешено использовать"

## Альтернатива:

Если нет возможности добавить реальные изображения, сайт будет работать с цветными плейсхолдерами.
